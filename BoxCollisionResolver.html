resolveBoxCollision(box) → {Manifold}

상자와 상자가 충돌했을 때 충돌깊이와 반작용방향을 반환한다.

          +-------+
+-----+   |       |
|  x  |   |   x   |
+-----+   |       |
          +-------+

   +--+   +---+    <-- 가로 길이의 절반
   +----------+    <-- 중심간의 거리

각 상자의 길이의 절반의 합이 중심간의 거리보다 작을 때에만 충돌이다. 이 때 충돌한 깊이는 각 길이의 절반의 합과 중심간의 거리의 차로 구해진다.
